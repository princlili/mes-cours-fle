# FLE Courses — Princlili

Teaching resources for French as a Foreign Language.

---

## Repository Contents

### Interactive Presentations (Reveal.js)

Slide decks built with [Reveal.js](https://revealjs.com/).

- [FLE Course Template](https://princlili.github.io/mes-cours-fle/cours-fle.html) — Base template for new lessons

**Slide navigation**
- Arrow keys or Space — move between slides
- `F` — fullscreen
- `Esc` — overview mode
- `S` — speaker notes

**Creating a new course**
1. Copy `cours-fle.html`
2. Replace content inside `[brackets]`
3. Save and test

### Course Plans (Kursplans)

HTML course schedules generated from Org-mode.

- [Français B1 (3/3) — Kursplan 2026](https://princlili.github.io/mes-cours-fle/kursplans/B1-3/kursplan_B1-3.html)

**Course index**
