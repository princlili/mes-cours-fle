# Export-Logik:
#   Nur Einträge mit :Lernende: werden in den HTML-Export einbezogen.
#   In der Emacs-Konfiguration oder im Export-Aufruf:
#   (setq org-export-select-tags '("Lernende"))
#
# Tag-Konventionen:
#   :Lernende:  → sichtbar im HTML-Export für Lernende (Séance-Titel, Schwerpunkt, Prüfung)
#   :Pruefung:  → Prüfung angekündigt — manuell setzen wenn geplant
#   :Lernatelier: → Lernatelier-Block (Mittwoch)
#   Abschnitt-Einträge tragen keinen Tag → unsichtbar im Export (kein :Lernende: = nicht exportiert)
#
# Prüfungsvorlage zum Kopieren:
# *** Prüfung :Pruefung:Lernende:
#     SCHEDULED: <YYYY-MM-DD Day HH:MM-HH:MM>
#     - Thema: [Einheit / Lektionen]
#     - Dauer: [45 min]
#     - Hilfsmittel: [keine / Wörterbuch / etc.]
